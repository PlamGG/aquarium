# คู่มือการใช้งานระบบจัดตารางสอน (School Scheduler)

เอกสารนี้อธิบายขั้นตอนการติดตั้ง, การตั้งค่า, และการใช้งานโปรเจกต์ระบบจัดตารางสอน

## สถาปัตยกรรมระบบ (System Architecture)

*   **Backend:** Python (Flask)
*   **Frontend:** JavaScript (React, Vite)
*   **Database:** SQLite (สำหรับ Development)
*   **UI Library:** Ant Design

## 1. การติดตั้ง (Installation)

ในการรันโปรเจกต์นี้ คุณจะต้องติดตั้ง **Node.js** (สำหรับ Frontend) และ **Python** (สำหรับ Backend) ก่อน

**ขั้นตอนการติดตั้ง:**

1.  **Clone a project:**
    ```bash
    git clone <repository-url>
    cd <project-folder>
    ```

2.  **ติดตั้ง Backend Dependencies:**
    สร้าง Virtual Environment (แนะนำ) และติดตั้ง library ที่จำเป็น
    ```bash
    # สร้างและเปิดใช้งาน virtual environment
    python -m venv venv
    source venv/bin/activate  # บน Windows ใช้ `venv\Scripts\activate`

    # ติดตั้ง dependencies
    pip install -r requirements.txt
    ```

3.  **ติดตั้ง Frontend Dependencies:**
    ```bash
    # เข้าไปที่โฟลเดอร์ frontend
    cd frontend

    # ติดตั้ง dependencies
    npm install
    ```

## 2. การรันโปรเจกต์ (Running the Project)

คุณจะต้องเปิด Terminal 2 อัน เพื่อรัน Backend และ Frontend แยกกัน

**Terminal 1: รัน Backend (Flask Server)**

```bash
# ตรวจสอบว่า virtual environment ถูกเปิดใช้งานอยู่

# รันเซิร์ฟเวอร์ Flask
flask run
```
*   Backend API จะทำงานที่ `http://127.0.0.1:5000`

**Terminal 2: รัน Frontend (Vite Dev Server)**

```bash
# เข้าไปที่โฟลเดอร์ frontend
cd frontend

# รันเซิร์ฟเวอร์สำหรับพัฒนา
npm run dev
```
*   คุณจะเห็น URL สำหรับเข้าถึงหน้าเว็บ (โดยปกติคือ `http://localhost:5173`)

## 3. การใช้งานระบบ (How to Use)

หลังจากรันโปรเจกต์และเข้าสู่หน้าเว็บแล้ว คุณจะพบกับเมนูหลัก 4 ส่วน

### 3.1. จัดการข้อมูลครู (Teachers)

*   **การดูข้อมูล:** หน้าแรกจะแสดงตารางรายชื่อครูทั้งหมดที่มีในระบบ
*   **การเพิ่มข้อมูล:**
    1.  คลิกปุ่ม **"Add Teacher"**
    2.  กรอกชื่อครูในหน้าต่างที่ปรากฏขึ้น
    3.  คลิก **"OK"**
*   **การแก้ไข/ลบ:**
    *   ในตารางของแต่ละครู จะมีปุ่ม **"Edit"** และ **"Delete"** ให้ใช้งาน

### 3.2. จัดการข้อมูลวิชา (Subjects)

*   **การดูข้อมูล:** แสดงรายวิชาทั้งหมดพร้อมรายละเอียด
*   **การเพิ่มข้อมูล:**
    1.  คลิกปุ่ม **"Add Subject"**
    2.  กรอกฟอร์มข้อมูลวิชาให้ครบถ้วน (ชื่อ, ระดับชั้น, จำนวนคาบ, ห้องพิเศษ)
    3.  คลิก **"OK"**

### 3.3. จัดการข้อมูลห้องเรียน (Rooms)

*   **การดูข้อมูล:** แสดงรายชื่อห้องเรียนทั้งหมด
*   **การเพิ่มข้อมูล:**
    1.  คลิกปุ่ม **"Add Room"**
    2.  กรอกฟอร์มข้อมูลห้องเรียน (ชื่อ, ประเภท, ความจุ)
    3.  คลิก **"OK"**

### 3.4. จัดตารางสอน (Schedule)

หน้านี้เป็นส่วนหลักของระบบ ประกอบด้วย 2 ฟังก์ชัน

1.  **การสร้างตารางสอน:**
    *   คลิกปุ่ม **"Generate Schedule"**
    *   ระบบจะเริ่มกระบวนการจัดตารางสอนในฝั่ง Backend โดยใช้ข้อมูลครู, วิชา, และห้องเรียนทั้งหมด
    *   **หมายเหตุ:** ในเวอร์ชันปัจจุบัน logic การจัดตารางยังเป็นแค่โครงสร้าง (placeholder) ผลลัพธ์ที่ได้อาจจะยังเป็นตารางว่าง

2.  **การดูตารางสอน:**
    *   หลังจากกด Generate แล้ว ตารางจะแสดงผลลัพธ์การจัดตารางสอน (ในอนาคต) ในรูปแบบตารางรายวัน/รายคาบ

## 4. โครงสร้างฐานข้อมูล (Database)

*   โปรเจกต์ใช้ SQLite และไฟล์ฐานข้อมูลชื่อ `app.db` จะถูกสร้างขึ้นโดยอัตโนมัติในโฟลเดอร์หลักของโปรเจกต์เมื่อมีการรัน Backend ครั้งแรก
*   หากต้องการเริ่มต้นฐานข้อมูลใหม่ทั้งหมด เพียงแค่ลบไฟล์ `app.db` แล้วรัน Backend ใหม่อีกครั้ง
